# Destapes Las Condes 24H - Landing Page

Landing page profesional para servicio de destape de cañerías urgente en **Las Condes**, Santiago, Chile. Diseño moderno, responsive y optimizado para conversión (llamadas y WhatsApp), con enfoque geográfico en Las Condes y sus barrios.

## 🚀 Características

- Diseño 100% responsive (mobile first)
- Header sticky con navegación suave por secciones
- Botones destacados de llamada y WhatsApp (número +56 9 6357 1251)
- Secciones: Hero, Problemas, Barrios, Servicios, Cómo funciona, Beneficios, Testimonios, Cobertura, CTA final y Footer
- **Nueva sección de barrios de Las Condes** con tiempos de llegada estimados
- Animaciones CSS suaves (entradas, hovers, flotantes)
- Imágenes reales del servicio (técnico, camioneta, trabajos)
- Optimizado para SEO y Google My Business (enfocado en Las Condes)
- Paleta de colores premium: teal oscuro (`#005F73`), dorado (`#E9C46A`) y naranja cálido (`#F4A261`)
- Tipografía moderna: Montserrat para títulos, Open Sans para cuerpo
- Código separado en HTML, CSS y JS para fácil mantenimiento

## 🛠️ Personalización

- **Colores**: Modifica las variables CSS en `:root` dentro de `styles.css`.
- **Barrios y tiempos**: Edita la sección `#barrios` en `index.html` para ajustar nombres y tiempos de llegada.
- **Teléfono y WhatsApp**: Reemplaza el número `+56963571251` en todos los enlaces (tel:, WhatsApp y footer).

## 📱 Uso

1. Clona o descarga este repositorio.
2. Abre `index.html` en tu navegador para previsualizar.
3. Sube los archivos a tu servidor web.
